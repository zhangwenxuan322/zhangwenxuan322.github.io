Objective-C中的Runloop

张文轩
2019-10-19

什么是Runloop

  • 高级一点的循环

  • 让线程在需要做事的时候忙起来，不需要的时候就休眠

Runloop的作用

  1. 保持程序运行，循环避免线程被销毁
  2. 处理App的各种事件(触摸、定时器)
  3. 节省cpu资源，提高程序性能

Runloop和线程的关系

  • 很明显的可以看出Runloop从 input source 和 timer source 接受事件，然后在线程中处理事件。
