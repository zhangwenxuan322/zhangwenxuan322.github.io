算法-二分查找法

2020-02-22

定义

二分查找也称为折半查找，每次都能将查找区间减半，这种折半特性的算法时间复杂度为 O(logN)。

关键

计算中值m，有两种方式：

  • m = (l + h) / 2
  • m = l + (h - l) / 2

第一种方式 l + h 可能出现加法溢出的情况，推荐使用第二种

正常实现方式

Input : [1,2,3,4,5]
key : 3
return the index : 2
public int binarySearch(int[] nums, int key) {
    // l 最低位 h 最高位
    int l = 0, h = nums.length - 1;
    // 进入二分查找循环体
    while (l <= h) {
        // 计算中值
        int m = l + (h - l) / 2;
        if (nums[m] == key) {
          	// m 个元素为查找值，返回结果
            return m;
        } else if (nums[m] > key) {
          	// m 个元素大于查找值，h 移动到 m - 1
            h = m - 1;
        } else {
            // m 个元素小于查找值，l 移动到 m + 1
            l = m + 1;
        }
    }
  	// 循环结束没有找到值，返回 -1 ，表示没有找到
    return -1;
}
